前端装机软件集合

  • 亲测
  • 前端

前端时间手贱，把台式机重装了下，结果，好吧，又要重新装软件了…

基本篇

sublime

编辑器，不解释，sublime2,sublime3
我个人是选了3。
顺便写上我安装的插件

  • Bracket Highlighter
    高亮插件
  • Emmet
    代码提示

    chrome firefox

    浏览器，不解释，调试的话，现在chrome比firefox好用了，特别是模仿手机那块，好使，当然还是要必须真机测试的，有时候话差上几个像素。

    koala

    开着它，就能愉快的用less咯，自动编译，yoyoyo。戳这里

    picpick

    我最主要是用来取色的，设置个快捷键方便的很，其实它功能还是很强大的，可以截图啊，图上弄点批注啥的。官网来一发
    有中文界面的，别担心。神奇的是，我打开的时候已经是中文界面了，要不要这么智能啊…

进阶篇

git

先上地址git
想用github不？那么git必须走起，啥，不知道github，赶紧百度，谷歌去吧

node.js

想愉快的玩耍吗？想的话还是装起来吧官网

装系统

一样在写了，顺便把我重装系统的内容也写下。

用了u天空助手，让我吐个槽，有些教程推荐的那些个u盘装机助手，简直就是个渣渣！
